SUPPLIES
--------

-----------------------------------------------------
Struct
-----------------------------------------------------

Suppiles
--------------------------------------
  code          | string | max len: 10
  name          | string | max len: 64
  unit          | string | max len: 10
  quantity      | string |

SuppilesList or SuppilesList
-----------------------------------------------------
  supplies      | Supplies       |
  left_node     | SuppliesNode * |
  right_node    | SuppliesNode * |

-----------------------------------------------------
Method
-----------------------------------------------------
Include:
  Object methods:
    - New and Destroy methods
  List methods:
    - New and Destroy methods
    - Logic methods
    - Insert methods
    - Get methods
    - Delete methods
-----------------------------------------------------

-----------------------------------------------------------------------------------------
Object methods
-----------------------------------------------------------------------------------------

New and Destroy methods
-----------------------------------------------------------------------------------------
NewSupplies     | Create a new Supplies
DestroySupplies | Destroy a Supplies

-----------------------------------------------------------------------------------------
List methods
-----------------------------------------------------------------------------------------

-----------------------------------------------------------------------------------------
New and Destroy methods
-----------------------------------------------------------------------------------------
NewSuppliesList                     | Create a new SuppliesList
DestroySuppliesList                 | Destroy a SuppliesList

-----------------------------------------------------------------------------------------
Logic methods
-----------------------------------------------------------------------------------------
IsInvoiceDetailListEmpty                 | Is InvoiceDetailList empty?
IsSuppliesCodeValid                      | Is Supplies code valid?

-----------------------------------------------------------------------------------------
Insert methods
-----------------------------------------------------------------------------------------
AddItemToSuppliesList               | Add a item to InvoiceDetailList

-----------------------------------------------------------------------------------------
Get methods
-----------------------------------------------------------------------------------------
GetItemInSuppliesListByCode            | Get InvoiceDetail in InvoiceDetailList by index

-----------------------------------------------------------------------------------------
Remove methods
-----------------------------------------------------------------------------------------
RemoveItemInInvoice                      | Remove item in InvoiceDetailList
RemoveItemInInvoiceDetailListByCode      | Remove item in InvoiceDetailList by index

-----------------------------------------------------------------------
Example
-----------------------------------------------------------------------
// Create new a SuppliesList
SuppliesList supplies_list = NewSuppliesList();

// Create new a Supplies
Supplies supplies = NewSupplies("abc", 0, 0.0f, 0.0f);

// Add InvoiceDetail to InvoiceDetailList
AddItemToInvoiceDetailList(supplies_list, supplies);

// Remove InvoiceDetail in InvoiceDetailList
RemoveItemInInvoice(supplies_list, supplies);

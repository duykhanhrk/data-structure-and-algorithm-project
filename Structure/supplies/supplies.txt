SUPPLIES
--------

-----------------------------------------------------
Struct
-----------------------------------------------------

Suppiles
--------------------------------------
  code          | string | max len: 10
  name          | string | max len: 64
  unit          | string | max len: 10
  quantity      | string |

SuppilesList or SuppilesList
-----------------------------------------------------
  supplies      | Supplies       |
  left_node     | SuppliesNode * |
  right_node    | SuppliesNode * |

-----------------------------------------------------
Method
-----------------------------------------------------
Include:
  Object methods:
    - New and Destroy methods
  List methods:
    - New and Destroy methods
    - Logic methods
    - Speciality methods
    - Insert methods
    - Get methods
    - Delete methods
-----------------------------------------------------

-----------------------------------------------------------------------------------------
Object methods
-----------------------------------------------------------------------------------------

New and Destroy methods
-----------------------------------------------------------------------------------------
NewSupplies                              | Create a new Supplies
DestroySupplies                          | Destroy a Supplies

-----------------------------------------------------------------------------------------
List methods
-----------------------------------------------------------------------------------------

-----------------------------------------------------------------------------------------
New and Destroy methods
-----------------------------------------------------------------------------------------
NewSuppliesList                          | Create a new SuppliesList
DestroySuppliesList                      | Destroy a SuppliesList

-----------------------------------------------------------------------------------------

-----------------------------------------------------------------------------------------
Logic methods
-----------------------------------------------------------------------------------------

IsSuppliesCodeValid                      | Is Supplies code valid?

-----------------------------------------------------------------------------------------
Insert methods
-----------------------------------------------------------------------------------------
AddItemToSuppliesList                    | Add a item to SuppliesList

Get methods
-----------------------------------------------------------------------------------------
GetItemInSuppliesListByCode | Get item in SuppliesList by code

-----------------------------------------------------------------------------------------
Remove methods
-----------------------------------------------------------------------------------------
RemoveItemInInvoiceDetailListByCode      | Remove item in SuppliesList by code

-----------------------------------------------------------------------
Example
-----------------------------------------------------------------------
// Create new a SuppliesList
SuppliesList supplies_list = NewSuppliesList();

// Create new a Supplies
Supplies supplies = NewSupplies("abc", 0, 0.0f, 0.0f);

// Add Supplies to SuppliesList
AddItemToInvoiceDetailList(supplies_list, supplies);

// Remove Supplies in SuppliesList
RemoveItemInSuppliesList(supplies_list, supplies);
